---
project: Competitive Time Machine
purpose: Visual diff tracker for e-commerce screenshots (Amazon, Target, Lowe's, Home Depot, Walmart, Menards)
stack: JS/HTML5/CSS3 (frontend), Node.js + Playwright (backend)
---

# Directory Structure
```
captures/[competitor]/[page]/[date]/{desktop.png, mobile.png, metadata.json?, analysis.json?, *-diff.png?}
src/{generate-index.js, analyze.js, diff.js}
viewer/{index.html, styles.css, viewer.js}
```

# Current Features
Multi-competitor sidebar | Date timeline | Side-by-side/diff/slider modes | Desktop/mobile toggle | Feature analysis panel | Viewport feature-status badges

- Metadata/analysis JSON is optional (gracefully skipped if missing)
